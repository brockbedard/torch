## Detailed Team Breakdowns

### RIDGEMONT BOARS — Power Spread
*"We're going to run the ball. You know it's coming. Stop us."*

**Offensive Philosophy:** Run-first, play-action passing. The OL fires forward. The RB is the star. When they do throw, it's off play-action — deep shots when you sell out to stop the run.

**Default Formations (weighted):**
- I-Form / Pistol: 40% of snaps
- Twins (2×1): 30%
- Shotgun 2×2: 20%
- Trips / Empty: 10% (change-up only)

**Play Card Pool — Offensive (weighted toward run):**

| Play Card | Type | Frequency in Draft Pool | Notes |
|-----------|------|------------------------|-------|
| Power | RUN | Very Common | Signature play. Lead blocker through B-gap. |
| Inside Zone | RUN | Very Common | OL combo blocks, RB reads the DT. |
| Toss/Sweep | RUN | Common | Gets RB to the edge with pullers. |
| Draw | RUN | Common | Fakes pass, hands off. |
| Play Action Post | ROUTE | Common | Fake run → deep shot. |
| Flat/Arrow | ROUTE | Common | Safety valve off play-action. |
| Corner Route | ROUTE | Uncommon | Deep sideline shot. |
| Slant | ROUTE | Uncommon | Quick pass when run is stuffed. |
| Screen | SCREEN | Rare | Only when defense is pinning ears back. |
| Go Route | ROUTE | Rare | Occasional home run ball. |

**QB Archetype:** Pocket passer or game manager. Doesn't need to run. Needs to sell the play fake and deliver deep when given the chance. Think: Jake Fromm, Mac Jones.

**RB Archetype:** Star of the offense. Physical, between-the-tackles runner. Patient. Finds the hole. Think: Derrick Henry, Najee Harris.

**WR Archetype:** Big-bodied possession receivers who can block on the edge for the run game and win contested catches on play-action. Think: Ja'Marr Chase.

**What They Look Like on the Field:** OL fires forward aggressively. RB hits the hole hard. When it's play-action, the fake is convincing — the RB carries out the fake while the QB delivers downfield. The animation should feel PHYSICAL — dots colliding, gaps opening.

**Defensive Identity:** Hard-nosed, run-stopping defense. Base Cover 3 with the LB filling gaps. Bend-don't-break zone. Not flashy, but disciplined.

### NORTHERN PINES WEREWOLVES — Spread Option
*"You have to defend the whole field. And the QB."*

**Offensive Philosophy:** Spread to run. The QB is an athlete who can beat you with his legs. Zone read is the base play — the QB reads the defensive end and decides give or keep. When the defense sells out to stop the QB, the receivers are open.

**Default Formations (weighted):**
- Shotgun 2×2: 35% of snaps
- Pistol: 25%
- Trips: 25%
- Empty: 10%
- I-Form: 5% (goal line only)

**Play Card Pool — Offensive (balanced, QB-run heavy):**

| Play Card | Type | Frequency in Draft Pool | Notes |
|-----------|------|------------------------|-------|
| Zone Read | RUN | Very Common | Signature play. QB reads DE. |
| QB Draw | RUN | Very Common | QB tucks and runs from shotgun. |
| Option/RPO | RUN | Common | Give/keep/throw triple threat. |
| Sweep/Jet | RUN | Common | Speed to the edge. Motion. |
| Slant | ROUTE | Common | Quick pass off zone read fake. |
| Mesh/Cross | ROUTE | Common | Crossing routes create confusion. |
| Seam | ROUTE | Common | Attack middle of the field. |
| Screen | SCREEN | Common | Gets athletes in space. |
| Go Route | ROUTE | Uncommon | Deep shot when defense cheats up. |
| Power | RUN | Rare | Only at the goal line. |

**QB Archetype:** Dual-threat. Fast. Decisive. The offense runs through him — literally. Think: Lamar Jackson, Jalen Hurts, Cam Ward.

**RB Archetype:** Quick, decisive, one-cut runner. Doesn't need to be huge — needs to make the first defender miss. Think: Bijan Robinson, Kenneth Walker.

**WR Archetype:** Speed receivers who can take the top off. Also need to be good blockers in the screen game. Think: Tyreek Hill, Jayden Reed.

**What They Look Like on the Field:** The QB is in motion — drop back, then tuck and run. The zone read creates a moment of tension where two dots (QB and RB) are both potential ball carriers. The defense has to commit. WRs run jet motion across the formation. It should feel FAST — quick snaps, dots accelerating.

**Defensive Identity:** Aggressive, blitz-heavy defense. Cover 1 man with the LB as a spy on the opposing QB. Force turnovers. Boom or bust.

### CRESTVIEW STAGS — Air Raid
*"We throw it early, we throw it often, and we throw it to people who can score."*

**Offensive Philosophy:** Pass-first. 4-second throw clock mentality — the ball comes out fast. Quick game concepts (slant, hitch, mesh) move the chains. Deep shots (go, post, corner) score touchdowns. The run game exists only to keep the defense honest.

**Default Formations (weighted):**
- Trips (3×1): 35% of snaps
- Shotgun 2×2: 30%
- Empty: 20%
- Bunch: 10%
- I-Form / Pistol: 5% (change-up only)

**Play Card Pool — Offensive (heavily pass-weighted):**

| Play Card | Type | Frequency in Draft Pool | Notes |
|-----------|------|------------------------|-------|
| Slant | ROUTE | Very Common | Quick 2-step delivery. Bread and butter. |
| Mesh/Cross | ROUTE | Very Common | Crossing routes at 5-6 yards. Man killer. |
| Go Route | ROUTE | Common | Vertical threat. Keeps defense honest. |
| Corner Route | ROUTE | Common | 15-yard break to the sideline. Cover 2 killer. |
| Seam | ROUTE | Common | Up the hash. Cover 2 killer. |
| Flat/Arrow | ROUTE | Common | Quick outlet. Yards after catch. |
| Screen | SCREEN | Common | Gets playmakers the ball in space. |
| Post | ROUTE | Common | Deep middle shot. |
| Draw | RUN | Uncommon | Change-up when defense drops 7 into coverage. |
| Zone Read | RUN | Rare | Occasional wrinkle. |

**QB Archetype:** Accurate pocket passer. Quick release. Reads coverage pre-snap. Doesn't need to run — needs to process fast. Think: Baker Mayfield, Patrick Mahomes (college), Kliff Kingsbury's QBs.

**RB Archetype:** Pass-catching back. Lines up in the backfield but runs routes out of it. Screens, flats, wheels. Think: CMC, Austin Ekeler.

**WR Archetype:** Route runners. Quick, shifty, get open fast. Multiple WRs who can win 1-on-1. Think: Wes Welker, Cooper Kupp, Davante Adams.

**What They Look Like on the Field:** Receivers are spread WIDE. The field looks open — lots of space between dots. The ball comes out fast — short, quick throws. Routes cross and mesh in traffic. When it's a deep shot, the WR is running away from the defender. It should feel SHARP — precise, surgical.

**Defensive Identity:** "Send everybody" pressure defense. Cover 0 blitz. Force the opposing QB to throw fast and hope for mistakes. High risk, high reward.

### BLACKWATER SERPENTS — Multiple / Pro Style
*"You don't know what's coming. Neither do we, until we see what you're in."*

**Offensive Philosophy:** Adaptive. This team doesn't have one base — they read the defense and call the play that exploits whatever they see. Heavy use of pre-snap motion to diagnose coverage. They'll run I-Form power one play and empty spread the next. They beat you with preparation, not athleticism.

**Default Formations (weighted):**
- Twins (2×1): 25% of snaps
- Bunch: 20%
- Shotgun 2×2: 20%
- I-Form / Pistol: 15%
- Trips: 15%
- Empty: 5%

**Play Card Pool — Offensive (balanced, concept-heavy):**

| Play Card | Type | Frequency in Draft Pool | Notes |
|-----------|------|------------------------|-------|
| Play Action | ROUTE | Very Common | Disguise is the whole point. |
| Power | RUN | Common | Physical run when defense spreads out. |
| Mesh/Cross | ROUTE | Common | Man coverage beater. |
| Flat/Arrow | ROUTE | Common | Quick safe throw to move chains. |
| Screen | SCREEN | Common | Change of pace. Misdirection. |
| Corner Route | ROUTE | Common | Deep shot off play-action. |
| Zone Read | RUN | Common | Option element. |
| Draw | RUN | Common | Deceptive run off pass look. |
| Post | ROUTE | Uncommon | Big play shot. |
| Trick Play | SPECIAL | Rare | WR pass, flea flicker, etc. The wildcard. |

**QB Archetype:** Smart, cerebral quarterback. Reads the defense pre-snap, adjusts, makes the right decision. Can make every throw. Think: Peyton Manning, CJ Stroud, Caleb Williams.

**RB Archetype:** Versatile back who can run between the tackles AND catch passes. A matchup problem. Think: Saquon Barkley, Alvin Kamara.

**WR Archetype:** Diverse receiving corps — one speed guy, one possession guy, one slot. Mismatches are the point. Think: LSU 2019 with Jefferson, Chase, and Marshall.

**What They Look Like on the Field:** Pre-snap MOTION. A WR motions across the formation. The RB shifts from one side to the other. The formation looks different every snap. When the ball is snapped, the play could go anywhere — run left, pass right, screen middle. It should feel UNPREDICTABLE — the defense is always wrong-footed.

**Defensive Identity:** Multiple looks defense. Cover 2 one play, Cover 0 blitz the next. Disguise pre-snap — show one thing, play another. Think: Nick Saban's defense.

## How This Creates Gameplay Differentiation

| Dimension | Boars | Werewolves | Stags | Serpents |
|-----------|-------|------------|-------|---------|
| **Draft pool** | Run-heavy cards dominate | QB run + balanced | Pass-heavy cards dominate | Evenly spread across types |

| **Strongest play type** | Power run, play-action | Zone read, QB draw | Quick pass, deep pass | Play-action, misdirection |
| **Weakest play type** | Deep passing | Between-the-tackles run | Running the ball | No clear weakness (master of none) |
| **Star player** | RB | QB | WR1 | Versatile flex player |
| **Field animation feel** | Physical, collision-heavy | Fast, QB in motion | Wide spacing, sharp throws | Motion-heavy, shifting |
| **Default defensive shell** | Cover 3 zone | Cover 1 man + spy | Cover 0 blitz | Multiple / disguised |
| **What beats them** | Spread them out, throw quick | Contain the QB, play zone | Run the ball, be patient | Execute fundamentals |

## Key Insight from Retro Bowl

Retro Bowl is the closest existing game to TORCH and has been wildly successful. But its most consistent criticism from hardcore players is that **every team runs the same plays.** There are no scheme differences. Picking the Saints vs the Packers is purely cosmetic — the formations and play calls are identical.

This is TORCH's competitive advantage. When you pick the Boars, your draft pool skews toward power runs and play-action. When you pick the Stags, you see more quick passes and deep shots. The team you choose changes the GAME you play, not just the colors on screen.

This is also how real college football works — picking Army means you're running the flexbone. Picking Texas Tech means you're airing it out. The scheme IS the identity.
