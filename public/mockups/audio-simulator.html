Click 🔊 INIT AUDIO first (browsers require user interaction to unlock audio). Then trigger individual plays or run a preset sequence. The HUD shows live crowd intensity, the target it's heading to, and the individual layer volumes (L/M/H crossfade). If smooth mode is off, behavior falls back to the old "linear Howler fade" style so you can compare.

This simulator uses the same crowd loop files as the game (/audio/crowd/crowd_{low,mid,high}.{webm,mp3}) and mirrors the fixed AudioManager logic. Once you're happy with the feel here, the same behavior is in src/engine/audioManager.js.
