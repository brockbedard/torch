# Season 2 — deferred scope

Per the Ember Eight bible, v0.40 ships single-game mode only. Season mode
(coach progression, ghosts, schedule, standings, multi-team identity) is
deferred to a later release.

This directory parks the in-flight exploration so it's not lost:

- **season-2-identity.html** — identity reveal mockup
- **season-2-logos.html** — team identity library
- **season-demo.html** — public-facing demo of season mode
- **teamsSeason2.js** — Season 2 team data structures
- **TORCH-SEASON-DEMO-PROMPT.md** — design prompt notes
- **TORCH-SEASON-FRAMEWORK.md** — season mode framework doc

When Season Mode v2 kicks off, start here.
